## HOST YOUR FIRE-X on VPS✳️ 
```
$ git clone `https://github.com/FireXbot/Fire-X`
$ cd firebot 
#install requirements

 
$ pip3 install -r requirements.txt 
#Create local.env 

$ python3 -m firebot 
```
